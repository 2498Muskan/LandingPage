# 🚀 StudyAI

### Learn Smarter. Study Faster.

StudyAI is a modern SaaS landing page concept for an AI-powered learning platform that helps students transform study materials into intelligent summaries, flashcards, quizzes, and personalized learning experiences.

Built with a clean and responsive design, the project focuses on delivering a professional SaaS user experience with smooth animations, dark mode support, and modern UI components.

---

## ✨ Highlights

* 🎨 Modern SaaS-inspired UI
* 🌙 Dark Mode Support
* 📱 Fully Responsive Design
* ⚡ Smooth Scroll Animations
* 📊 Animated Statistics Counter
* 💬 Testimonials Section
* 💰 Pricing Plans
* ❓ FAQ Section
* 📬 Newsletter Subscription
* 🎯 Call-to-Action Components

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* Bootstrap Icons
* AOS (Animate On Scroll)

---

## 🎯 Goal

The purpose of this project is to create a professional SaaS landing page that demonstrates modern frontend development practices, responsive layouts, engaging animations, and user-friendly design principles.
